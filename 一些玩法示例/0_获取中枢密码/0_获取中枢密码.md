# 自动获取小米中枢网关的登录密码


## 原理
1. 发post请求
2. 解析返回
3. 将返回直接粘贴到输入框中自动验证并跳转

2. 快捷指令open极客版url，同时把**验证码**用?的方式传入参数
3. safari浏览器安装userscript脚本，负责将验证码自动填入
